# Requirements

## Operating System
- Windows 10 (22H2 or newer)
- Windows 11 (all supported builds)

## Permissions
- Administrator privileges are required for most repair, network, optimization, and security modules.

## Runtime Dependencies
- Native Windows tools (`DISM`, `SFC`, `CHKDSK`, `NETSH`, `PowerShell`, `WEVTUTIL`, `ROBOCOPY`).
- PowerShell 5.1+ for advanced diagnostics/report tasks.

## Storage
- Minimum 200 MB free disk space for logs, reports, and backups.
